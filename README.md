# Sales Dashboard

## Overview
An Excel dashboard built to analyze sales performance 
and turn raw data into actionable business insights.

> Early project focused on practicing data analysis 
> and dashboard design using Excel.

## Tools & Techniques
- Pivot Tables — summarizing and slicing data
- KPIs — tracking key business metrics
- Dynamic Charts — visualizing trends over time
- Conditional Formatting — highlighting performance gaps

## What the Dashboard Shows
- Total revenue and sales volume
- Top-performing products or categories
- Monthly and quarterly trend lines
- Performance highlights via color-coded formatting

## Notes
This was one of my first data projects — focused on 
transforming raw sales records into a clean, readable 
Excel dashboard.
