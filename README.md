# 📊 Superstore Sales Analysis Dashboard

## 📖 Overview
This project analyzes Superstore sales data to uncover business insights related to sales performance, shipping modes, and product categories.  
An interactive dashboard was built using Excel to help understand trends and support decision-making.

---

## 🛠 Tools & Technologies
- Microsoft Excel  
- Pivot Tables  
- Charts & Data Visualization  
- Slicers (Filters)  

---

## 📊 Key Insights
- Analysis of delayed orders compared to same-day and first-class shipping  
- Sales performance across different regions  
- Sales distribution by sub-category  
- Sales trends over time  
- Interactive filtering to explore data dynamically  

---

## 📁 Dataset
- Superstore dataset
